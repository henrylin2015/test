## 功能说明

	主要练习git的一些操作，命令等。

	1. 主要命令
	   
	   * git init
	   
	   * git add file
	   
	   * git commit -m""
	   
	   * git push origin master
	   
	2. 命令说明
	   
	   * git init 初始化git库
	   
	   * git add file  添加文件
	   
	   * git commit -m "" 提交文件，说明提交的这些文件意义。
	   
	   * git commit --amend  -m "" 意思是说，当上次提交提交时，一个文件没有删除或是修改干净(如一些测试代码没有删除),用这个命令会把这次修改了得文件提交到上次修改的文件中，少一次commit 描述
